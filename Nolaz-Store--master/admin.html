Setup Instructions

  1. Create a Google Sheet with columns: id, name, price, description, category, image, status
  2. Deploy the provided Google Apps Script as a web app
  3. Set up Cloudinary account for image hosting
  4. Update configuration in admin.js
  5. Publish your Google Sheet as CSV for the main store
